# 🩺 Breast Cancer Detection ML App

## 📌 Overview
End-to-end machine learning web application for predicting breast cancer diagnosis using multiple classification models.

## 🚀 Features
- SVM, Random Forest, Logistic Regression comparison
- Confusion Matrix visualization
- ROC-AUC curve
- Interactive Streamlit UI
- Dataset download option
- Deployed on Streamlit Cloud

## 🛠 Tech Stack
- Python
- scikit-learn
- Streamlit
- NumPy
- Matplotlib
